Grid Flexbox SASS

Bootstrap 4 inspired CSS3 Flexbox implementation of Responsive Grid.

Features:

  • Integrated ellipsis
  • Stretch columns to full width
  • Full height
  • Vertical align center (.col-align-middle)
  • Simple Responsive show & hide system
